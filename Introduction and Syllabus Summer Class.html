Graphing with GGPlot Introduction and Syllabus

Welcome to Graphing with GGPlot

The main purpose of this class is to learn how to create robust plots in R using GGPlot. GGPlot is a widely used R package that adopts a modern approach to graphing that makes plots more extensible and reusable and it also has one of the largest, and most active, R communities. The class will also focus on good programming practices and include a variety of topics concerning plotting in R including but not limited to pattern recognition (grep), RGB color codes and Unicode characters, gradients, and quantiles.

This is a 1-credit class and the expectation is that you will spend about 60 hours overall on the class -- this time includes the class meetings.

Prerequisites

This class is designed for students that have some programming experience, whether in R or another language (e.g., Python, JavaScript, C++). The experience needed is about the equivalent of one semester of programming – or an understanding of functions, if-else statements, and for loops.  You will need a laptop with the newest versions of R (3.6.3 or 4.0+) and RStudio (1.2+).

Class Structure

The class will be styled similar to a flipped classroom where students will go through the lessons before class and the class period will be used to present the material in alternatives ways, teach extensions to the material, and to allow student to work together on the lesson applications and their class projects.  It is expected that students will come to class having read through the lesson and made a good effort on the lesson application.

Lessons

The course has 10 lessons divided up over the 7 weeks, the exact order of lessons might change during the course.

Week 1:

RStudio Projects: Set up software and the programming environment for the class, go over the programming environment

Week 2:

Components: Getting data from a CSV file, scatterplots, breaking down GGPlot instructions, mapping elements

Plot Styles: Plot themes, styling components on a plot, labels and title, regression lines

Week 3:

Date Objects: Modifying data columns, creating CSV files, line plots, modifying the paneling and background of the plot area

Histograms:date formatting, if() and which() statements with multiple conditions, histograms, adding lines, faceting

Week 4:

Boxplots 1: quantiles, boxplots, factoring variables, annotating plot, outliers

Boxplots 2: ordering variables, grouping boxplots, RGB colors, faceting boxplots, legends

Week 5:

Barplots: variables types, filling bars by variable, reordering data frame, legend style changes

Week 6:

Text Plots: scatterplot with text, gradients, pattern recognition (grep), legend styling

Week 7:

Multipanel Plots: more pattern recognition, subsetting values, conditions on vectors, multiple plots on a canvas

Lesson Application

Every lesson has an Application section at the end that will ask you to apply the skills taught in the lesson to your own script.  The applications are not graded and the answers are provided on the GitHub page for the class.

    Copying codeblocks

    Double-clicking on a codeblock will select all the text in the codeblock and copy it to the clipboard -- you can then paste it into RStudio (or any other text editor). 

    # the next two lines should be at the top of all your scripts
    rm(list=ls());                       
    options(show.error.locations = TRUE);
    # create three variables: d, t, and v
    # give d and t values and use them to calculate v
    d = 100;
    t = 20;
    v = d/t;
