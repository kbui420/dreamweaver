Introduction and Syllabus

Welcome to Graphing with GGPlot

The purpose of this class is to introduce students to the graphing package in R called GGPlot2 and the modularization techniques used by GGPlot2 to create plots.  GGPlot2, developed by Hadley Wickham, is a powerful data visualization tool used to make publication-quality plots and it has one of the largest, and most active, R communities.  GGPlot2 differs from other graphing packages by separating a plot into its component parts, making the code for the plot more reusable and extensible. 

 

This class uses RStudio and is designed for students who have at least a basic background in programming -- the equivalent of one semester of R, or any similar programming language (e.g., Visual Basic, JavaScript, C++).

 

Graphing with GGPlot is a self-paced and asynchronous online class. The amount of material in the class is about the same as a 1.5 credit, one semester class and students are expected to complete the class within three months.  There are no regularly scheduled office hours but you can email or call the instructor to ask questions or set up a meeting.  All contacts will receive a response within one business day.

Class Structure

The course has 10 lessons:

  1. RStudio Projects: Set up software and the programming environment for the class, go over the programming environment
  2. Components: Getting data from a CSV file, scatterplots, breaking down GGPlot instructions, mapping elements
  3. Plot Styles: Plot themes, styling components on a plot, labels and title, regression lines
  4. Date Objects: Modifying data columns, creating CSV files, line plots, modifying the paneling and background of the plot area
  5. Histograms:date formatting, if() and which() statements with multiple conditions, histograms, adding lines, faceting
  6. Boxplots 1: quantiles, boxplots, factoring variables, annotating plot, outliers
  7. Boxplots 2: ordering variables, grouping boxplots, RGB colors, faceting boxplots, legends
  8. Barplots: variables types, filling bars by variable, reordering data frame, legend style changes
  9. Text Plots: scatterplot with text, gradients, pattern recognition (grep), legend styling
  10. Multipanel Plots: more pattern recognition, subsetting values, conditions on vectors, multiple plots on a canvas
